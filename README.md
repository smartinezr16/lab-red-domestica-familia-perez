# Lab-1 Red Doméstica de Streaming Familia Pérez - Interconexión 

## Descripción

En este laboratorio se diseñó e implementó una red doméstica funcional utilizando Cisco Packet Tracer.  
El objetivo fue simular la conexión del hogar de la familia Pérez a un servicio de streaming, integrando dispositivos cableados e inalámbricos dentro de una misma red LAN/WLAN.

Se configuró direccionamiento IP dinámico (DHCP) para los clientes y direccionamiento IP estático para el servidor, además de validar la conectividad mediante los protocolos ICMP y HTTP.
